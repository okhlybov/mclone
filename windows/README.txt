Mclone is a utility for offline file synchronization utilizing Rclone as a backend for doing actual file transfer.
 
This Windows installer bundles Mclone, Rclone and Ruby runtime (required by Mclone) in one easy to use package:

* Mclone :: https://github.com/okhlybov/mclone
* Ruby 3.0.1 :: https://rubyinstaller.org
* Rclone 1.55.1 :: https://rclone.org

The installer provides command line launcher `mclone.exe` and optionally exposes it to the PATH environment variable.
